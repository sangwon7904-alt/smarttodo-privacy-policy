# Privacy Policy

**Last updated:** July 2026

## SmartTodo Privacy Policy

SmartTodo respects your privacy.

### Information Collection

SmartTodo does **not** collect, store, or transmit any personal information.

### Data Storage

All Todo data is stored **locally on your device**.

No data is uploaded to any server.

### Data Sharing

SmartTodo does **not** share user data with third parties.

### Permissions

SmartTodo may request the following permissions:

- Notification permission (to remind you of scheduled tasks)
- Exact alarm permission (to deliver reminders at the correct time)

These permissions are used only for the app's core functionality.
